\itemprimerapagina
{Resumen}
    {
        Aquí debe completar un resumen en español de 200 palabras como máximo, que contenga una clara indicación de los objetivos, alcance y resultados.
    }

\itemprimerapagina
{Palabras clave}
    {
        Después del resumen se debe incluir un número máximo de 6 palabras clave separadas de un · (e.g. transporte público \and estaciones \and diseños \and experimentos).
    }

\itemprimerapaginacursiva
{Abstract}
    {
        Además, en la misma página, se debe incluir un abstract (en inglés) de similar longitud que el resumen en español. 
    }
\itemprimerapaginacursiva
{Keywords}
    {
        Después del abstract se debe incluir un número máximo de 6 keywords separadas de un · (e.g. public transport \and stations \and designs \and experiments) 
    }
